# 小说创作助手 (PGA Admin) 产品功能概览 (产品视角)

本文档从产品经理及最终用户的视角展示“万象星际”小说生产系统的功能矩阵与当前运行状态。

> 世界层级、独立 Agent 工作流、禁止假数据、禁止非世界全量查询、真实 requests 验收测试等详细需求见：[世界层级与 Agent 迭代工作台产品需求文档](./docs/product_world_hierarchy_requirements.md)。

## 1. 核心看板 (首页)
**核心价值**: 实时掌握系统“心跳”，确保创作环境稳定无虞。
- **服务健康监控**: 自动检测后端 API、数据库及向量索引，确保创作链路通畅。
- **创作日志流**: 透明化展示 AI Agent 的思考与执行过程，方便追踪创作进度。
- **多供应商接入**: 动态展示当前使用的 LLM 模型（如 Gemini, GPT-4 等）及其配置。

## 2. 世界观知识库 (设定中心)
**核心价值**: 沉淀并可视化小说的宏大设定，确保内容逻辑自洽。
- **分级实体管理**: 以直观的树状结构组织种族、地理、势力等核心设定。
- **语义检索系统**: 集成深度语义搜索，支持模糊意图匹配，告别简单的关键词过滤。
- **可视化图谱**: 动态生成**关系力导向图**，一眼看清复杂的人物/势力网。
- **思维导图预览**: 将庞杂的设定分类映射为**思维导图**，辅助大框架梳理。
- **零延迟编辑**: 支持在浏览器内直接修正设定，即时同步至 AI 知识库。

## 3. 实体草案审查 (质量闸门)
**核心价值**: 严控 AI 自动生成设定的质量，确保“设定不崩、逻辑不乱”。
- **待审列表**: 自动汇集 AI 在创作过程中自发衍生出的新概念、新角色。
- **人工干预闭流**: 提供批准、拒绝、及“继续打磨”操作，确保每一条正式入库的设定都经过人工确认。

## 4. 小说大纲管理 (逻辑锚点)
**核心价值**: 确定故事走向，为 AI 创作提供高维指引。
- **多版本管理**: 支持多套大纲提案的对比与版本迭代。
- **人机协同生成**: 引入 HITL (Human-in-the-loop) 工作流，Agent 构思过程中实时同步进度，并支持用户在中途深度介入审核与修改。
- **分布式同步**: 批准的大纲可自动转化为分布式 SKILL 协议，指引后续章节创作。

## 5. 章节创作管理 (执行终端)
**核心价值**: 极致的 AI 辅助写作体验，实现“人机共创”。
- **流式对话创作**: 通过对话形式指挥 Writing Agent 进行章节撰写。
- **断点反馈机制**: 用户可在 Agent 创作过程中间随时插入意见，Agent 会根据反馈动态调整后续走向。
- **所见即所得编辑**: 对生成的正文进行精细化的人工修饰与二次创作。
